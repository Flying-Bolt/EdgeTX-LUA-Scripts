# EdgeTX LUA Scripts
EdgeTX Widgets für Horus X12S (480×272px)

## Widgets

| Widget | Funktion |
|--------|----------|
| **BattCheck** | Akku-Überwachung (Telemetrie) |
| **Clock** | Uhr-Anzeige |
| **Counter** | Zähler |
| **Druck** | Luftdruck-Anzeige |
| **Gauge** | Analoges Instrument |
| **RPM** | Drehzahl-Anzeige |
| **Temp** | Temperatur-Anzeige |
| **Timer2** | Timer |
| **VAMeter** | Spannung/Strom-Anzeige |

Alle Widgets unterstützen mehrere Zonengrößen (2x4, 2x2, 2x1, Fullscreen).
